Getting Started

  1. Edit monitoring/content/index.content.html to modify the contents of this page.
  2. Create a new page by making page-name.content.html file in the monitoring/content/ directory.
  3. Read up on available components in the Component Showcase.
  4. Create your own component templates by following the Custom Components Guide.
  5. Create your own custom API endpoints by following the Custom API Endpoints Guide.

Goals to keep in mind

  • Monitor your product live
  • Validate your product's content and functions
  • Create feedback loops by publishing user and usage statistics
